# Panda Copilot 提案升級版

## 一、重新定位：從銀行內部工具改成第三方新創平台

原本 GreenFlow Copilot 偏向銀行內部工作台。  
升級後的 **Panda Copilot** 應明確定位為：

> **由第三方新創團隊提供的綠色授信協作平台**，站在銀行端與企業端之間，負責資料整合、補件引導、報告生成、案件比較、流程加速與貸後追蹤。

這樣的定位有三個好處：

1. **更容易跨銀行複製**
   - 不被單一銀行內部系統綁住
   - 可做成 SaaS / 白標 / API 模式

2. **更容易吸引企業端**
   - 企業不必每家銀行重做一次資料包
   - Panda Copilot 可把資料整理成多銀行可用格式

3. **更有商業化空間**
   - 可向銀行收取訂閱費
   - 可向企業收取進階送件與顧問服務費
   - 可與設備商、ESCO、顧問、保險、租賃合作抽成

---

## 二、核心主張

### 平台名稱
**Panda Copilot**

### 品牌敘事
- Panda 代表穩健、可信、友善、容易被記住
- Copilot 代表不是取代銀行或企業，而是協助兩端更快完成綠色授信流程

### 核心一句話
> Panda Copilot 幫企業把資料變成可送件的綠色授信資料包，幫銀行把案件變成可閱讀、可比較、可追蹤的前審與貸後管理工作流。

## 四、Panda Copilot 的完整功能升級

## A. 企業端入口

### 1. 智慧送件入口
企業可依合作銀行指定格式上傳：
- 台電帳單
- 電子發票
- 財報摘要
- 銀行往來摘要
- 專案計畫書
- 設備報價單
- 碳盤查資料
- ISO 文件
- 供應鏈問卷
- 還款來源說明

### 2. 缺件即時檢核
平台自動告訴企業：
- 哪些是必要文件
- 哪些是加分文件
- 哪些資料會影響綠色授信適配度
- 目前距離可送件還差什麼

### 3. 企業版送件包生成
平台可生成：
- 企業綠色授信申請摘要
- 專案 CAPEX / 節能效益摘要
- 碳費暴露與改善空間摘要
- 專案簡報版摘要
- 管理層簡報用重點摘要

---

## B. 銀行端入口

### 1. 銀行前審 Memo
自動生成：
- 公司基本輪廓
- 專案用途
- 財務韌性
- 文件完整度
- 風險紅旗
- 建議產品
- 補件清單
- 建議下一步

### 2. 雙公司 / 同產業比較
銀行可快速比較：
- 綠色效益
- 財務韌性
- 文件完整度
- 風險程度
- 是否適合優先推進

### 3. 授信條件與 KPI 草案
平台可產出初版：
- 資金用途約束條件
- 專案里程碑
- 年度節能目標
- 減碳目標
- 綠電占比目標
- 貸後追蹤指標

### 4. 貸後追蹤月報
平台可延伸為：
- 專案執行進度
- 實際用電變化
- 再生能源占比變化
- ESG / 問卷更新情況
- 達標率與異常提醒

## 七、與 qwen3.6 的整合亮點

Panda Copilot 搭配 qwen3.6 時，提案可以強調：

1. **多角色報告生成**
   - 企業摘要
   - 銀行 Memo
   - 補件清單
   - KPI 草案

2. **長上下文整合**
   - 可以把多份文件整理成一致口徑輸出

3. **自然語言問答**
   - 使用者不必自己翻所有文件
   - 直接問：「這家公司最適合哪一類產品？」
   - 直接問：「還缺哪些關鍵文件？」

4. **結構化 + 生成式雙引擎**
   - 核心分數與規則先算好
   - LLM 再負責把結果轉成專業可讀內容
   - 降低幻覺風險
